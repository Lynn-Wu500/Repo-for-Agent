# Repo-for-Agent

**500 · Finance Prompt Studio**

一个面向财务流程的交互式 Prompt 提示词生成器与提示词库。选择业务场景、AI 扮演的角色、约束条件与输出格式，即可一键生成并复制可直接粘贴到大模型（ChatGPT / Claude / 文心一言等）的中文提示词。

## 在线访问

GitHub Pages: **https://Lynn-Wu500.github.io/Repo-for-Agent/**

（首次开启 Pages 后约 1–2 分钟生效。）

## 功能

- 🛠️ **交互式生成器**：场景 / 角色 / 任务 / 背景 / 准则 / 约束 / 输出格式 / 语气，实时拼装提示词
- 🤖 **财务助手（演示）**：基于内置的「星辉零售 2026年6月」虚构财务数据即时问答，可真实计算营收、毛利率、费用超预算、应收账龄、现金流、报销稽核、增值税等。纯本地运算，不联网、不消耗任何 API、无密钥泄露风险。
- 📋 **一键复制**：复制后直接粘贴到大模型使用
- 📚 **精选提示词库**：12 条，覆盖报销、对账、结账、报表分析、税务、预算、审计、成本、发票、往来核销等财务场景，可「用作模板」载入生成器
- ➕ **贡献入口**：通过 GitHub Issue 提交你的提示词

## 本地预览

直接用浏览器打开 `index.html` 即可，无需任何依赖或构建。
