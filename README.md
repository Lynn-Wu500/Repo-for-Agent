# Repo-for-Agent

**500 · Finance Prompt Studio**

一个面向财务流程的交互式 Prompt 提示词生成器与提示词库。选择业务场景、AI 扮演的角色、约束条件与输出格式，即可一键生成并复制可直接粘贴到大模型（ChatGPT / Claude / 文心一言等）的中文提示词。

## 在线访问

GitHub Pages: **https://Lynn-Wu500.github.io/Repo-for-Agent/**

（首次开启 Pages 后约 1–2 分钟生效。）

## 功能

- 🛠️ **交互式生成器**：场景 / 角色 / 任务 / 背景 / 准则 / 约束 / 输出格式 / 语气，实时拼装提示词
- 🤖 **财务助手（演示 + 联网评审）**：默认基于内置「星辉零售 2026年6月」虚构数据本地问答（不联网、不消耗 API）。输入**联网咒语**可解锁真实大模型，用来**评审你在上方生成的提示词**并给出改写建议。
- 📋 **一键复制**：复制后直接粘贴到大模型使用
- 📚 **精选提示词库**：12 条，覆盖报销、对账、结账、报表分析、税务、预算、审计、成本、发票、往来核销等财务场景，可「用作模板」载入生成器
- ➕ **贡献入口**：通过 GitHub Issue 提交你的提示词

## 联网模式（口令解锁真实大模型）

助手支持解锁真实大模型来评审提示词。安全设计：**真 key 从不以明文出现在源码里**，只以「口令加密后的密文」形式存放（AES-GCM + PBKDF2）。用户输入正确口令后，浏览器在本地解密还原 key 再调用。

> ⚠️ 这是纯静态客户端方案：**知道口令的人可以在浏览器里看到还原后的 key**，仅适合给自己 / 信任的人使用。要对不特定人开放请改用后端中转。

### 如何配置 / 换 key

1. 用浏览器打开仓库里的 `encrypt-key.html`，填入你的**有效 key** 和**口令**，点生成。
2. 把生成的两段分别替换 `index.html` 中的 `NET.ENCRYPTED_KEY` 与 `NET.SPELL_HASH`。
3. 按厂商改对 `NET.ENDPOINT` 和 `NET.MODEL`（页面里有注释与示例）。
4. 打开页面，在助手输入框输入口令即可解锁；随后粘贴提示词发送，模型会按评审模板输出。
